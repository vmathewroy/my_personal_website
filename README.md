# Daily Habit Tracker

A web application for tracking daily habits and monitoring progress over time. Built with vanilla JavaScript and Supabase for database management and authentication.

## Features

- **Habit Logging**: Record daily habit completions organized by custom categories
- **Point System**: Each habit has a configurable point value to track your progress
- **Date Navigation**: View and log habits for any date, with easy navigation between days
- **Visual Analytics**:
  - Daily bar chart showing points earned per category
  - Monthly progress chart displaying cumulative performance
- **Secure Authentication**: Email-based magic link login powered by Supabase
- **Responsive Design**: Clean, dark-themed interface optimized for desktop use

## Tech Stack

- Frontend: HTML, CSS, vanilla JavaScript
- Database: Supabase (PostgreSQL)
- Charts: Chart.js with ChartDataLabels plugin
- Authentication: Supabase Auth (email OTP)

## Table Schemas

### habit_categories
```sql
create table public.habit_categories (
  id bigint generated by default as identity not null,
  category_name text not null default ''::text,
  constraint habit_categories_pkey primary key (id),
  constraint habit_categories_category_name_key unique (category_name),
  constraint habit_categories_id_key unique (id)
) TABLESPACE pg_default;
```

### habits
```sql
create table public.habits (
  id bigint generated by default as identity not null,
  name text not null,
  active boolean null default true,
  default_points bigint not null,
  category bigint not null,
  constraint habits_pkey primary key (id),
  constraint habits_category_fkey foreign KEY (category) references habit_categories (id) on update CASCADE on delete CASCADE
) TABLESPACE pg_default;
```

### habit_log
```sql
create table public.habit_log (
  id uuid not null default gen_random_uuid (),
  date date not null,
  habit bigint null,
  constraint habit_log_pkey primary key (id),
  constraint habit_log_habit_fkey foreign KEY (habit) references habits (id) on update CASCADE on delete CASCADE
) TABLESPACE pg_default;
```
